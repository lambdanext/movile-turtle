# movile-turtle

A simple emulation of the Logo drawing system 

## Usage

```
(use '[movile-turtle.core :only [show!]])

(show! [[:left 30] [:fwd 200] [:left 180] [:fwd 100] [:right 60] [:fwd 100]])
```
